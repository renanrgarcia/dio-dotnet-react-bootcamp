# Loops

- for

```cs
for (int i = 0; i < 10; i++) {
  // do something
}
```

- while

```cs
while (condition) {
  // do something
}
```

- do while

```cs
do {
  // do something
} while (condition);
```

- break: stop the loop
- continue: skip the rest of the loop

## Extras

- Envoirment.Exit(0): exit the program
- Console.Clear(): clear the console
